# wordcount

> Really minimalistic word counter.

![WordCount Screen Recording](./assets/screencap.gif)

## Note

This was made because I wanted to learn WebAssembly. Pls no hate for the bloat.

## Development

Requirements:

- Git
- NodeJS
- `pnpm`
- Rust Toolchain + Cargo
- `wasm-pack`

To Develop:

1. `git clone https://github.com/neelkarma/wordcount.git`
2. `pnpm i`
3. `pnpm build-core`
4. `pnpm dev`

To Build:

1. All the steps but the last in To Develop
2. `pnpm build`
